Detailed Description

Text styling attributes

Note
THREAD SAFETY: This struct uses bitfields which share storage. While copying a Style is safe, concurrent modification of the same Style instance from multiple threads is NOT safe. The bitfield implementation means that writing to one field (e.g., bold) could theoretically affect adjacent bitfields in a data race scenario.

Always use value semantics (copy Styles) rather than shared mutable references when working across thread boundaries. For single-threaded TUI usage, this is not a concern.
